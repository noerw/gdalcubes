Detailed Description

Data structure for defining packed data exports (dividing and adding data values by a scale and offset value respectively, before conversion to smaller integer types during the export.

Member Data Documentation

std::vector<double> gdalcubes::packed_export::nodata

A vector of new nodata values. Must have size() == 1 or size() == nbands If the vector contains only one element, the same nodata value is assumed for all bands.

std::vector<double> gdalcubes::packed_export::offset

A vector of per-band offset values. Must have size() == 1 or size() == nbands If the vector contains only one element, the same offset value is assumed for all bands.

std::vector<double> gdalcubes::packed_export::scale

A vector of per-band scale factors. Must have size() == 1 or size() == nbands If the vector contains only one element, the same scale factor is assumed for all bands.
